# 第七章 宁静的世界
```
心灵越平静，我们取得成就、获得影响、走向善良的权力就越大。
```

富有智慧的人总是能够获得心灵上的平静。人们可以通过长期耐心的自我控制来实现心灵上的平静。

人类是由思维而进化的生物，对自己的认识越深刻，我们的心灵就会越平静。当我们对生活中那绝对有效的原 因与结果法则有了更清醒的认识之后，我们就会不再满腹牢骚、终日抱怨、担忧和难过，相反，我们会变得气定神闲、优雅从容。

只有内心平静的人，才能知道如何管理自己、适应别人；而那些内心充满躁动的人则会为这些人身上的精神力量感到敬畏，就会变得更加依赖他们。

心灵越平静，我们取得成功、获得影响、走向善良的权力就越大。就连最普通的推销员，在拥有了更大的自我控制和平衡能力以后，他的生意也会日益兴隆。因为人们总是更乐意和令人愉快的人打交道。

坚强平静的人总是会有人爱，有人尊敬。因为对于周围的人来说，他们就像是沙漠里的绿阴或者风暴中的避风港，总是带给人们一种安静、平和、安全的感觉。

谁不想过上一种甜美、稳定、宁静的生活呢？这样他们就不用再担心外界的变化，因为拥有这些品质的人内心总是充满甜蜜、宁静和镇定。宁静是一种近乎完美的品格，也是我们的最后一课；宁静是生命的花儿盛放，是灵魂丰收的果实。它和智慧一样宝贵，比黄金更值得渴望。与宁静的生活相比，金钱真是微不足道——如果能够永远生活在风暴之外，生活在永恒的宁静里，那金钱又能算得了什么呢？

可在现实生活中，有多少人在把自己的生活弄成一团乱麻啊！他们脾气火暴，吵吵闹闹，骂骂咧咧，破坏了周围的平静生活。相比之下，我们遇到的品质完美，性格平静的人真是太少了！

如果不能够控制自己的怒火，你的命运就会变得更加曲折，如果不能克制自己的悲哀，就会变得每日烦乱不堪，如果任由自己的内心充满焦虑和疑惑，你就会整天坐立不安。只有那些真正富有智慧、思想纯净而懂得节制的人，才能够过上安静祥和的生活。

年轻的读者们啊，如果你们的灵魂正在遭受折磨，请记住这些话：

在生活的海洋里，祝福的岛屿在微笑，理想的阳光海滩在等你来。
坚定地把握思想的舵。
灵魂的船上靠着指挥官——他指挥，但他睡着了。
叫醒他。
自我克制能够带给你力量；正确的思想将使你变得无比精巧；而平静将使你变得有力。
对你的心说：“安静！”
