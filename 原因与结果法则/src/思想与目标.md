# 第四章 思想与健康
```
确立你的目标，
然后无畏地向它进发，
你就能创建自己的未来——
```

在一个人把思想和目标联系起来以前，他不会有任何有意义的成就。因为对大多数人来说，他们的思想就像是一块枯树皮，整日在生活的海洋上随波逐流。他们在过着一种漫无目的的生活，而对于这样的人来说，迎接他们的必将是灾难和毁灭。

生活没有中心目标的人很容易陷入生活中琐碎的忧愁、恐惧和自我怜悯等情绪里面。而这些情绪又都肯定（虽然会以不同的方式）会带来失败，最终带给人们苦闷和失落。

我们需要有一个合理的目标，并应该马上动手实现它。我们应该使这个目标成为生活的重心。对于不同的人来说，这个目标的具体形式是不同的，它可能表现为一种精神上的追求，也可能是一个具体的物质目标；但无论是什么，我们都应把思想的力量集中到这些早已设定的目标上面来。我们要让这个目标成为自己的最高职责，把自己的全部身心都投入到实现它的过程当中，而不要放任自己整天只考虑那些空洞、无聊的事情。

这是我们实现自我控制和真正的集中思想的唯一有效的方法。即便在实现目标的过程当中，我们会一再地失败——我相信，在我们克服自身的诸多缺点之前，这些失败也是必要的——但在这个过程当中，我们终究会获得一些更加美好的品格和力量，从而为我们走向成功作好充分的准备。

对于那些还没有为自己的生活确立一个伟大目标的人来说，你应该集中思想，完美地完成自己当前的任务，对自己所做的每一件事尽到职责——无论那职责有多微小。只有这样，你的思想才能集中，而你的精力和技能才能得到进一步的发展，一旦完成了这些，你就可以更加容易地获得成功。

如果你觉得自己的意志还不够坚强的话，那没有关系，只要你能够认识到自己的软弱，并立即采取实际行动，尽自己的最大努力，不断尝试，不断进步，你最终就会变得坚强。

一个虚弱的人总是能够通过认真耐心的锻炼增强体质，思想也是一样，思想软弱的人也可以通过正确的锻炼让自己的思想变得坚强。

马上开始有目的的思考吧，进入那些坚强者的行列，不要惧怕失败，它们是帮助你走向成功的必要因素，运用周围的一切条件为你服务，坚定地思考，无畏地尝试，直到走向成功。

克服怀疑和恐惧

确定目标之后，你的下一步就是要在头脑里确定一条通往成功的道路，要采用最直接的方式，既不向左也不向右。抛掉那些怀疑和恐惧吧，它们只会让你在实现目标的过程中多走弯路，浪费你的时间、精力和信心。

怀疑和恐惧不能，也永远不会成就任何事情。它们总是导致失败。一旦让它们溜进你的大脑，就会对你所有的目标、精力和做事的力量产生消极的影响。

无论你的目标是什么，都一定要对自己的力量充满信心。那些喜欢怀疑和恐惧的人，一定要努力消除掉这些情绪，否则你就会在自己的人生道路上遭遇更大的挫折。

征服了怀疑和恐惧，你就征服了失败。一旦变得坚定勇敢，你就会立刻充满力量，就可以勇敢面对任何困难并明智地克服它们；你就可以为自己确立明确的目标，朝着它们坚定地前进，并最终取得成功。

那些敢于无畏地朝向目标的人总是会充满创造性，这样他们就会变得更高更强；而相比之下，思想飘忽不定的人则远不及他们。所以，现在就采取行动，朝着明确的目标前进吧，因为只有这样的人才能有意识地、聪明地利用自己的精神力量。
