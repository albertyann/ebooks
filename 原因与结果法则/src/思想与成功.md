# 第五章 思想与成功

```
一切成就无论是经商、学术，还是在精神世界里，都是有着确定方向的结果
```

你一切成就和失败都是自己思想的直接结果。在一个合理的世界里，失去平衡意味着完全的毁灭，所以你个人必须对此负有绝对的责任。你的所有软弱和坚强，你的纯洁和不纯洁，最根本的原因都在于你自己；造成这一切的人就是你自己；而且你也不要指望有其他人能够帮助你改变，要想做到这一点，你只能靠自己。你的环境也是由你自己造就的，而不是别人。你的喜怒哀乐都是从你自己的内心生发出来的。你思故你在，一个人之所以会存在，最根本的原因就是因为他在不断地思考。

如果软弱的人在思想上拒绝帮助，再坚强的人也无法帮助他，软弱的人必须自己坚强起来，他们必须通过自己的努力去获得他们梦想的坚强。只有他们自己能够改变自己当前的处境。

无论是压迫别人的人，还是那些被人压迫的人，他们的现状往往是由他们自己造成的。一个人，只要拥有了完善的知识，他就能看到被压迫的人和压迫别人的人思想上的缺点；同样，只要有了纯洁的爱，一个人就会在被压迫的人和压迫别人的人身上看到双方都在承受着苦难，谁都不应该受到谴责。总而言之，真正拥有同情心的人就能够真正理解这些人内心的苦闷。

只有那些已经征服了软弱，赶跑了自私的人，才是真正自由的，他们不会压迫别人，也不会受到别人的压迫。

只有提升了思想，我们才能不断进步，并有所成就。否则的话，我们就只能一直生活在软弱、可怜和痛苦之中。

在成就任何事情，即便是很世俗的事情之前，我们都必须首先控制好自己的思想。当然，我们不必为了成功放弃所有的自私，但我们必须放弃一部分。因为如果我们一味地放纵自己的思想的话，我们就无法清晰地思考，也不能作出有条理的计划；如果不控制好自己的思想的话，我们就不能发现和发展那些隐藏的资源，所以无论做什么，我们最终都只会失败。如果还没有开始有效控制自己的思想，我们就不能够控制自己所遇到的情况，也就不能够有力地承担重大责任。

不作出一定程度的牺牲，我们就不可能取得进步和成功，无论是从物质还是从精神的角度来说，越能够控制自己的思想集中自己的注意力、坚定自己的决心，我们就能够取得越大的成功。思想越高尚，我们就会越正直；越能坚持理想，我们就能够取得越大的成就，获得越持久的成功。

## 什么样的人不能获得成功

贪婪欺诈、邪恶的人是不会取得真正的成功的，尽管有时他们会辉煌一时；而诚实、宽宏、有德行的人终究会得到帮助，并最终实现自己的理想。一切年代的伟大导师都以不同的形式得出了这个结论，为了更好地证明这个道理，我们就必须坚持使我们的品德更加高尚。

要想在智力上取得巨大的成就，我们就必须集中思想，认真钻研知识以及生活中所有那些美好的事物。虽然有些人在为着自己的虚荣心不断努力，但他们的成就并不是虚荣心的结果，这些人之所以会成功，完全是他们长期纯洁无私的努力的结果。

如果一个人始终不懈地坚持自己的理想，他就必将能够达到成功的顶峰。而那些一直让自己思想保持纯净无邪的人，就一定会变得明智、高贵、幸福而富有威望，这是毫无疑问的。

人们所取得的任何成就都是他们不断努力的结果。在确定了目标，并在实现目标的过程中不断控制自己的思想之后，我们就能够不断进步，并最终实现自己的目标；而与之相反，那些思想混乱、懒惰、缺乏自控的人，最终必将一事无成。

当然，如果不能一直坚持那些美好的思想的话，即便我们取得了短暂的成功，也可能会很快堕落到傲慢、自私腐败当中，并最终走向失败。一切成就，无论是经商、学术，还是在精神世界里，都是有着确定方向的结果。虽然具体的目标不同，但所有的成功者都必须遵循同样的法则。

要想小有成就，你就得作出小的牺牲；要想成果辉煌，你就必须作出更大的牺牲。
