# 第三章 思想与健康
```
人的身体是脆弱的，
它很容易就会受到思想的影响，
思想健康的人，身体就会健康，
反之亦然。
```

人的身体很容易就会受到思想的影响。如果一个人的思想不健康，他很容易就会染上疾病，身体就会慢慢垮掉；与此相反，如果一个人思想快乐美好的话，这个人就会变得更加年轻、美丽。

我们前面讲过，人能够主导自己周围的环境，不仅如此，人还能够主导自己的身体健康状况。有的人之所以会生病，根源就在于他有着不健康的、病态的思想。如果一个人总是生活在对疾病的恐惧里的话，他就会成为真正的病人，原因非常简单：当一个人心里充满焦虑的时候，他的这种情绪就会在整个身体上表现出来，疾病也就会随之而来。

相比之下，当一个人思想健康、纯洁、快乐的时候，他的身体就会变得更强壮健美。人的身体是脆弱的，它很容易就会受到思想的影响，思想健康的人，身体就会健康，反之亦然。
当一个人的思想变得不再纯洁的时候，他们的血液里面就会生出致病的因子；而当一个人心灵纯良洁净的时候，他的生活和身体都会变得积极健康。反之，如果一个人的思想肮脏、不纯洁的话，他的生活和身体也就会变得不纯洁。思想是行动、生活和言行的源泉；洁净了源泉，一切都将变得洁净。
有人相信饮食可以改变一个人的健康状况，可对于那些不肯改变思想的人，再好的饮食也不会起到任何作用。如果我们的思想真正纯洁的话，就不会想吃那些不纯洁的食物；自身思想纯净的人不需要考虑那些有害的微生物。
如果你想让自己的身体完美健康的话，请首先护卫好你的头脑；而如果你想让自己的身体焕发出新生的光彩的话，就一定要先美化你的头脑。一个内心充满邪恶、嫉妒、失望、沮丧的人是不会拥有健康的身体的。有些人整天一张苦脸，这不是偶然的，它最根本的原因就在于这个人的思想。如果一个人内心充满愚蠢、痛苦和骄傲的话，他的脸上很快就会爬满皱纹。

我的朋友当中就有很多这样的例子。比如说我认识一位96岁的女士，她的面容纯洁而明亮，就像少女一样。我还认识一个远未到中年的男子，他的脸上全是皱纹，俨然已经成了个老头子。与此相对应的是，那位女士总是充满阳光，而那位男子却整天不安，牢骚不断。

思想与健康的关系其实非常简单，想想看，如果不让空气和阳光自由地进入房间，你就不可能拥有一个甜美健康的居所；同样，如果想让你的身体健康、明亮、快乐、宁静的话，那你就必须首先让自己的心灵充满喜悦、善意和宁静。

在那些长者的脸上，我们总是可以看到一条又一条的皱纹，有的来自强健纯洁的思想，而其他的则来自于消极的、不健康的思想。那些内心充满正直的长者仿佛是正在徐徐西下的落日，充满平静、柔和、成熟。

我最近见到过一位临死的哲学家。从表面上看来，他一点都不老。在这一生当中，他一直过的都是一种甜美宁静的生活，如今，他也以同样的方式死去，对他来说，即便死亡也充满了恬静、平和。

乐观向上的思想是最好的医生；而和蔼与善良也是治愈病痛的最佳良药。如果一个人一直让自己的内心充满疑虑、愤世嫉俗和嫉妒的话，他就会一直生活在自己造的监狱里。而如果一个人能够怀着善意和希望看待一切，能够耐心地发现一切事物中的美好，并从中学习的话，他就会让自己的生活充满安静、祥和。
