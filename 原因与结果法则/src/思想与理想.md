# 第六章 思想与理想
```
那些怀着美好梦想，
并有着高贵理想的人，
终有一天会实现它。
```

梦想家是这个世界上最宝贵的财富。正是因为有了这些梦想家，我们的世界才变得如此美好。

人类不能忘记梦想家；不能让他们的梦想褪色、死亡。他们总是生活在自己的梦想当中，而且他们坚信，这些梦想终有一天会变成现实。

作曲家、雕塑家、画家、诗人、预言家，他们都会对自己所在的整个世界产生影响。这个世界因为他们的存在而变得美丽；没有他们，劳作的人类必将毁灭。

那些怀着美好梦想和高贵理想的人，终有一天会实现自己的梦想和理想。哥伦布梦想着能够找到另一个世界，结果他找到了；哥白尼相信地球并不是这个世界的中心，他揭示并证明了自己的梦想；那些伟大的修行者们梦想到了一片美丽无瑕、完善宁静的精神世界，他们进入了那个世界。

珍惜你的梦想和理想吧，它们是能够震撼你心灵的音乐，是能够塑造你头脑的美景，它可以为你带来巨大的动力和精神上的愉悦。真诚对待你的梦想，它最终就能变成现实。

有梦想才能有方向，有目标才会有成功。我们应该只去满足那些最卑下的欲望，而不理会我们内心那最纯洁的理想吗？这不是宇宙的法则，这样的人终将一无所获。

要有高贵的梦想，因为你必将能够把自己的梦想变成现实。相信自己，大胆地去梦想，梦想中的美景就是你的未来，你的理想将永远指引着你前进的方向。

梦想更好的人生吧！

古往今来，所有最伟大的成就在刚开始的时候都只是一个梦想。在长成一棵参天大树之前，橡树只是一粒种子；在还没有孵化、长大、高飞的时候，小鸟只能在蛋壳里等待。同样，在所有那些美好的理想成为现实之前，它们都只能是一个个听起来甚至有些荒谬的梦想。但无论它在开始时听起来有多么荒谬，千万不要因此而放弃梦想，因为梦想是现实的幼芽，如果一个人没有了梦想，他就不可能创建自己美好的未来。

你暂时可能正处于逆境之中，但无论如何，只要你拥有梦想并努力实现它，你就能不断改善自己的处境。因为正如我们前面所说的那样，当一个人的内心发生改变的时候，他所处的环境也必将随之发生改变。

从前有一对年轻的男女，他们都生活在贫困之中。为了生活，他们不得不长期在条件恶劣的车间里工作。他们没受过教育，身上没有任何优雅的素质，但他们每时每刻都在梦想着能够改变自己的命运，能够得到更好的东西。他们梦想着自己能够变得更聪明、更优雅、更美丽；他们梦想着能够过上一种完美的生活；能够进入大图书馆看书，扩大自己的视野。一想到这些，他们内心就充满了不安，而这种不安又成为了他们不断努力的动力。就这样，他们利用一切空闲时间和手段，不断地学习，让自己进步。

很快，他们的思想改变了，他们对自己周围的环境也有了更新的认识，他们感觉这破旧的车间已经不能再容纳他们了，于是他们准备像扔掉一件外套一样脱离自己当前的生活，并继续为着这个目标而不断努力，就这样，当机会来临的时候，他们永远离开了原来的位置，彻底地改变了自已的命运。


几年以后，他们成为了完全不同的人，他们用自己的方式，掌握了影响世界的权力，他们的一句话能够改变许多人的
生活，人们开始对他们充满期待，并开始按照他们的指引规划自己的生活，就这样，这一对年轻人终于实现了自己年
轻时的梦想。


你，年轻的读者，无论你的梦想是什么，只要不是空洞的大话，你就必将能够实现它，因为在你的内心深处，你悄悄热爱的东西会一直吸引着你。


无论目前的处境如何，你都可以拥有朝着梦想进发的权利。如果你的梦想高尚的话，你就会变得越来越高尚；而如果你的梦想卑下的话，你也就就会变得越来越卑下。梦想有多大，你的抱负就有多大，而你的成就也就会有多大。

任何成功都是不断努力的结果

斯坦顿·克尔凯·戴维斯曾经写下这样的美丽句子：

也许你现在还是个记账的，但现在你将走出门，重新寻找一个新的世界，你将发现自己站在众人面前——笔还夹在耳朵上，指头上还有墨迹——而就在彼时彼地，你的灵感将喷涌而出……

也许你是个来自乡村的牧羊人，有一天，你漫游到城市，你说话大大咧咧，口无遮拦；但你对自己充满信心，你游逛到大师的工作室，过了一段时间，他会告诉你：“我已经没有什么可以教你的了。”就这样，你成了大师，回头想想，你之所以会来到城市，之所以会成为大师，原因就在于你在牧羊的时候产生的一个梦想。

那些缺乏考虑、无知、懒惰的人，他们只知道谈论运气、幸福和机会，却从来没有想过这些东西背后的原因。看见有人发财了，他们会说：“这家伙真走运！”看见有人成为了著名学者，他们感叹：“这家伙命真好！”注意到有些人的高贵品格，能够对人们产生巨大影响，他们又会说：“这些家伙真是太幸运了！”

可另一方面，他们却从没考虑过这些人所默默承受的考验、经历的失败、作出的挣扎；他们没有看见这些人为着梦想所作出的牺牲、付出的努力，以及确立的信念；他们没有想到，这些人是在克服了无数看起来似乎不可征服的障碍之后，才最终实现了自己心中的梦想。

他们不知道这些人所经历的失败和心痛；只看见了他们所享受的成功和喜悦，并称之为“幸运”；他们没看见这些人在漫长的路途中所经受的艰辛，只看见了他们愉快地到达了目的地，于是他们就说这些人“好运气”。

无论做什么，我们都需要付出努力，而我们所得到的结果，往往就反映了我们所付出的努力的大小。所谓的天赋、力量、才智……都是一个人不断努力的结果。

珍视你的梦想和理想吧！它们将建造你的生活，你将成为自己梦想中的样子。
