# 第二章 思想与环境
```
人类的意志是一种看不见的能量，

它能够开辟出通向任何目标的道路，

即便有花岗岩的高墙阻拦。
```

<strong>我们可以把人的头脑比作一个花园，你可以聪明地耕耘它，也可以放任不管一一但无论如何，它里面都一定会生长些什么。</strong>如果你种下一粒花草的种子，它就会长出美丽的花草；如果你什么也不做，它里面就会长出无用的杂草，并日益滋生。

就像园丁在花坛里培植花木，除去杂草，种植自己喜欢的花草一样，你也要照管自己思想的花园，清除那些错误的、无用的、不纯洁的思想，并努力培养出那些正确的、有用的、纯洁的思想。在这个过程当中，你迟早会发现，你是自己灵魂的园丁，自己生活的向导。不仅如此，你还会发现，在自己的内心深处，存在着一些绝对的思想法则，并且你还会日益清晰地看出，你的思想和头脑会对你的性格、环境以及命运产生巨大的影响。

人的思想和品格是合于一体的。而且由于一个人的品格总是能够通过他所处的环境和境遇反映出来，所以你的外在环境也就总是和你内心深处的思想一致。当然，我并不是说在任何情况下，你的处境都能反映你所有的品格，但你必须意识到，你当前所处的环境确实与你的思想当中的一些关键因素有着十分密切的关系，所以说，要想改变自己的处境，你首先就必须改变这些关键性的因素。

我们前面说过，在一个人的生长过程当中，他必然遵循一些绝对的法则。你的思想将对你的性格起决定性的作用，所以当你安排自己的生活时，你总是遵循着自己的思想所指引的方向，这是一条顛扑不破的法则。无论你是个同环境格格不入的人，还是一个凡事都容易满足的人，这个法则都同样有效。

<strong>每个人都是可以不断进步的，人的每一步发展都是一个学习和成长的过程，无论在生活中遇到任何挫折，你都可以从中学到一些教训，而一旦你学会了这些教训，挫折也就会相应地消失，更好的境遇也就会随之而来。</strong>

可现实生活中，有很多人总是在埋怨周围的环境，如果是这样的话，你就会一直受到环境的局限，永远不可能突破眼前的困境，真正彻底改变自己的命运。与此相反，一旦你意识到自己的体内蕴藏着一种可以创造一切的力量，你就能以全新的眼光来审视自己周围的环境，成为环境的主人，并最终按照自己预想的方向来改变它。

所有能够进行自我审视、自我控制的人都知道，人的处境实际上是由自己的内心思想所决定的，因为他们注意到，当他们的思想状况发生改变的时候，他们的处境也就会随之发生相应的变化。这是绝对真实的，当你诚恳地改进自己的性格缺陷的时候，你就能迅速取得进步，并使自己的生活发生一系列变化。

## 原因与结果法则

<strong>人的心灵总是会吸引那些它会不自觉地接纳的东西，无论是它喜爱的还是它害怕的。</strong>它能够攀抵理想的高峰，也可以跌人恐惧的谷底。好的、有益的思想结出好的果实，而坏的、有害的思想则结出有害的果实。

一个人所面对的外在环境能够随着这个人的内在思想不断发生改变，而无论是愉快的还是不愉快的，这些外在条件都会影响到一个人的最终收获。所以我相信，每个人所得到的都是自己思想和行为的结果，而在播种与收获的过程当中，你所经历的所有痛苦和喜悦都能让你学到一些教训和经验。

跟随你自己内心深处的欲望、抱负和思想吧！向着它们所指引的方向发展，最终你就会从外部环境中收获相应的果实。辛勤耕耘，你就必然能得到收获；不断调整，你就一定能够取得进步。这样的法则放之四海而皆准。

有的人流落街头或者是被投人监狱，这并不是命运或环境在跟他作对。因为一个内心纯洁的人绝对不会仅仅由于外在的力量而突然犯罪；事实上，罪恶的思想早已经在他的内心深处潜生滋长，时机一到，这些思想所积聚的力量就会爆发出来。所以说，<strong>环境并不能造就人，它只是在反映这些人自己内心深处的思想。</strong>

如果我们内心深处没有蕴藏着邪恶的思想，周围的环境并不能使我们做出邪恶的举动，我们也就不会因此而承受所有随之而来的痛苦；同样，如果不是坚持培养美好的品质和理想，我们也就不能拥有美好的品德和快乐。因此，作为自己思想的主人，我们可以主宰自己的命运，造就和改变我们所处的环境。

<strong>当我们出生的时候，我们的灵魂是独立的。在我们渐渐成长的过程当中，它逐渐吸引那些与自己趣味相同的东西，并将这些东西反映在自己那纯洁或邪恶、有力或软弱的思想与性格当中。</strong>

<strong>我们所吸引的并不是我们希望得到的东西，而是那些和我们本质相同的东西。</strong>我们所有的奇思怪想、幻想和野心在每一步都会遭到阻挠，但我们内心深处的思想和欲望却始终坚持着自己的方向。那主导命运的力量就在我们的内心深处；它就是我们所有行为的最根源的动力。所以事实上，一直以来，唯一局限我们的力量就是我们自己，而主宰我们命运的就是我们自己的思想和行为：卑下的思想和行为会限制乃至毁灭我们；而高贵的思想和行为也能够解放并造就我们。

## 首先，从根本上改变

希望和祈祷并不能帮助我们得到自己想要的东西，要想得到这些东西，我们就要通过自己的努力去赚取它。<strong>只有当我们的希望和祈祷与我们的思想和行动一致时，我们所有的要求才会得到满足。</strong>

那么，既然如此，我们为什么还要“与环境作斗争”呢？在我看来，通过与环境作斗争，人们实际上是在不停地反抗一个并不存在的后果，可另一方面，导致这个后果的原因就在我们内心深处。这个原因有时会表现为有意识的邪恶，有时又会表现成无意识的软弱；但无论以什么样子出现，它都会顽固地阻碍拥有它的那个人的努力，成为他取得进步的一个绊脚石。

我们大多数人都急于改善自己所处的环境，却不愿意改善自己，这样的人往往都会以失败告终。而另一方面，如果我们能够诚恳地进行自我剖析并不断调整自己，我们就能达到任何目标。就算我们把追求财富当作自己的唯一目标，我们也必须准备好作出巨大的个人牺牲一一而<strong>对于那些想要过上稳定而平衡的生活的人来说，他们就需要作出更大的牺牲。</strong>

让我们看几个例子：

有这么一些穷困潦倒的人。一方面，他们非常希望能够改善周围的环境和家里的条件，但另一方面，他们却总是在偷懒，不好好工作。不仅如此，他们甚至觉得自己很有理由欺骗雇主，因为他们感觉这些人付给他们的工资很少。在我看来，这些人根本不明白那些甚至是最基本的道理。我认为，这些人不仅不能够脱离困境，而且，如果他们一味地放纵自己的胡思乱想，整天只想着偷懒和欺骗的话，他们最终只会把自己引向更糟糕的处境。

还有这么一些非常有钱的人，他们总是对自己的饮食不加任何节制，结果开始生病，深受其害。可问题是，这些人愿意花大笔钱来治疗自己的病痛，却不愿意放弃贪吃的习惯。就这样，他们一方面希望身体健康，另一方面却又毫无节制地山吃海喝。这些人也不可能拥有健康的体魄，因为他们没有认识到健康生活的基本原则。

还有这么一些雇主，他们用欺诈手段克扣工人的工钱，希望获得更大的利润。这样的人最终也只能走向破产的命运。可就是这些人，当在声誉或金钱上遭遇重大损失时，他们就会抱怨环境，而没有意识到自己才是唯一该对自己的境遇负责的人。

我举这些例子，只是为了说明，人们总是一一虽然经常在毫无意识的情况下一一造成自己境遇的最主要的原因，一方面，这些人有着非常美好的目标；可另一方面，他们却总是在希望通过一些与目标截然相反的方式来实现自己的目标。这样的例子几乎可以无限列下去，但我觉得在这里并没有这个必要。如果愿意的话，你们可以在自己的头脑和生活里反省自己的思想和行为，但我必须告诉你们，无论处于怎样的境遇，我们都不应该简单地埋怨外部环境。

## 原因与结果法则放之四海而皆准

但我们所处的环境常常是非常复杂的，而我们的思想却又常常隐藏得很深，而且虽然我们自己能够了解自己的精神状态，其他人却很难通过我们的某些行为对我们的思想作出判断。有的人给人感觉非常诚实，可他却一直生活在贫困当中；而另一个人可能让人感觉是个狡诈的小人，却得到了大笔的财富。因此常常有人根据这些现象得出结论，认为诚实的人总是会倒霉，只有狡诈的人才能够取得成功。事实上，这是一种非常武断而肤浅的判断，因为它认为狡诈的人几乎总是一无是处，而诚实的人一定就品德高尚。

可在有了更深人的认识和更广泛的经验之后，我们就会知道，这种想法是错误的。因为事实上，被认为是狡诈的那个人往往也会有些令人敬佩的好处；而那个被认为诚实的人也会有一些坏毛病。——即使这些毛病可能微不足道。所以说，我们前面谈到的原因与结果法则在这里同样适用：诚实的人因为诚实的思想和行为而得到相应的好处，但也要为自己的那些毛病而承担相应的后果；同样，狡诈的人因为自己的狡诈而遭受了惩罚，但他的那些优点也会给他带来收获。

每个人都有虚荣心，所以我们总是愿意相信自己所受的每一分苦难都是因为自己的好德行——可事实上，如果仔细剖析一下自己的话，我们就会发现，自己思想深处还存在着许多缺点，而我们所承受的一切苦难，实际上正是这些缺点所造成的结果。在彻底改正这些缺点之前，我们就不能简单地认为自己的苦难是由好德行而引起的。所以说，生活中的原因与结果法则依然存在，而且它也绝对不会混淆善恶。在知道这一点之后，我们就会意识到，其实生活中的一切都是在始终不变地遵循着这个绝对的因果法则，<strong>我们所遭遇的一切都是由我们自己所造成的。</strong>

<strong>好的思想和行为不会产生坏的结果；而坏的思想行为也不会产生好的结果</strong>。中国有句谚语：“种瓜得瓜，种豆得显。"这个道理不仅适用于自然界，也同样适用于精神和道德世界。

如果一个人感到痛苦，那肯定是因为他内心深处存在着某种错误的思想。因为当人遭受痛苦的时候，就说明他的内心和他所处的环境产生了矛盾。在我看来，痛苦的唯一的、最大的作用就是净化、燃尽一个人内心深处那些无用和不纯洁的东西。纯洁的人不会受苦。因为当一切杂质都被驱除之后，自然就没有必要继续用苦难对他进行锤炼。

人们之所以会感到痛苦，那是因为自己的精神世界出现了不和谐。相反，当人们的精神达到和谐的时候，他们就会更多地感到愉悦。要想实现心理上，而不是物质上的愉悦，一个人就必须有正确的思想；反而言之，当一个人的思想错误的时候，他就必然会在精神上，而不是物质上，遭受痛苦和煎熬。我们经常可以看到，有些人非常有钱但却非常痛苦，而有些人虽然贫穷但却很快乐。所以说，只有懂得正确运用财富的人才能够在拥有财富的同时得到真正的幸福；而对于那些处境艰难的穷人，如果他们一味认为自己的贫穷是由于生活的不公平所造成的话，他们就只能继续在贫穷中挣扎。

<strong>贫穷和放纵同样是痛苦的两个极端。这两种状态都是人们精神世界不平衡的结果</strong>。如果我们的生活不快乐、不健康、不富足的话，那我们的生活就是不正常的；相对来说，当我们感到快乐、健康富足的时候，就说明我们真正地实现了精神世界与外部环境之间的和谐。

停止抱怨和咒骂吧！让我们循着那绝对不变的原因与结果法则重新开始自己的生活，只有这样，我们才能真正感到快乐、健康、富足。当我们开始不断调整自己的心态，掌握并利用生活中的因果法则的时候；当我们不再一味地把自己的处境怪罪到别人头上，而开始塑造健全健康的思想的时候；当我们停止和环境作对，并开始利用它来帮助自己取得更快的进步的时候，我们就会实现真正的快乐健康、富足。

## 了解宇宙的法则

世间万物流转，但它们都要遵循一定的法则；而我坚信，生活中最本质的力量也必然是正义和正直。因此，要想真正地认识世界万物的规律，首先我们就必须承认这些法则的存在，并循着这些法则来真正地认识我们所处的世界。而在这个过程当中，我们会发现，当我们改变自己对待周围的人和事物的态度的时候，这些人和事物也会开始变得对我们有利。

相信每个人都会有这样的经历，而且我们可以很容易地通过反思和自我剖析来发现这个道理。我们知道，当我们能够真正地从根本上改变我们的想法的时候，我们的生活条件就会发生巨大的变化。虽然很多人相信别人根本不会了解自己的思想，可事实上，你会很不自觉地把它表现为自己的习惯，而你的习惯则会进一步影响和造就你的处境。

一方面，如果一个人思想卑下的话，他就很可能会酗酒、发牢骚，并最终陷人贫穷和痛苦。有的人思想混乱懒惰，他在生活中就会表现得毫无条理，而且经常给人一种疲惫不堪的感觉，这样的人就会渐渐地对自己周围的人产生怀疑；有的人内心充满恐惧怀疑、犹豫，在日常工作和生活当中，他们就会显得非常软弱，而且做事经常半途而废，最终这样的人就会陷人失败、贫困和堕落；有的人思想懒惰，他们在生活中就会表现得邋遢、虚伪，这样的人最终会变得肮脏、贫困；有的人内心充满憎恨，他在生活中就会经常埋怨别人，并表现出很强的暴力倾向，这样的人最终会给周围的人造成伤害，自己也会因此而锒铛人狱。那些内心自私的人呢？他们会在生活中表现得极具掠夺性，而他的这种性格又会给他带来令人痛苦的后果。

而另一方面，如果一个人有着美好的思想的话，他就会在生活中表现得善良而宽厚，因此他的周围就会充满一片和善。有些人的思想非常富有建设性，这种人在生活中就会表现得非常能够容忍，而且能够很好地控制自己，他的这种性格又会使他的生活充满宁静与和平；有的人富有勇气、具有很强的自立性和决断力，这样的人最终就会取得成功，过上富足而自由的生活；有的人精力充沛，他就会让人感觉非常整洁勤勉，这又会给他带来无尽的愉悦；有的人内心温柔，充满谅解，这样的人会在与人相处的时候表现得非常善解人意，他也就会在自己的周围创造出一个安全而健康的环境；有的人内心充满爱，具有无私奉献的精神，这样的人就会经常为了别人而忘记自己的利益，从长远来看，他就必然能取得长久而确定的成功，并获得真正的富。

## 你所处的环境是反映你思想的万花筒

一系列的思想坚持下来，不论好坏，都肯定会对我们的品格和环境造成影响。任何人都不可能随心所欲地选择环境，但我们却能够选择自己的思想，并因此间接而确定地对自己的周围环境产生影响。

大自然和我们在一起，而且总是帮助我们满足我们最渴望得到的东西，当机会来临的时候，我们的思想一一无论它是建设性还是破坏性的一一就会以一种最快的方式呈现出来。

一旦我们开始主动停止那些消极的、毁灭性的思想，整个世界都会向我们敞开大门，并时刻准备向我们伸出援助之手。当我们把那些软弱病态的思想从自己的身体内驱逐出去之后，幸运之神就会随时跳出来帮助我们。而一旦我们鼓励那些美好的思想，我们的命运就不会再充满崎岖坎坷，我们的苦难也自然就会随之结束。如果说世界是一个巨大的万花筒，那它所展现的不同颜色的组合就是我们时刻变动的思想的反映。

```
你总能左右自己的命运：

那些总是埋怨“环境”的人注定要遭遇失败，

事实上，这种埋怨是毫无道理的，停止埋怨环境，开始从自身努力吧，你的灵魂将从此变得自由。

从此以后，无论在什么情况下，你的成功将不再依靠机遇，

你将底脱离环境的影响，并使它为你的仆人，为你所控制。

人类的意志是一种看不见的能量，

它能够开辟出通向任何目标的道路，

即叟有花岗岩的高墙阻拦。

不要为眼前暂时的困扰而忧虑，

耐心等待吧；

当你的思想开始发生改变时，

一切都将如你所愿。
```
