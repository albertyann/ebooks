# Sikll

Skill 作为一种 AI 的接口方式，已经发布了好几个月了，然后因为各种原因，并没有关注。这周重新完整的梳理各种 AI 的咨询，发现 skill 是最近被提及比较多的技术，看了文档之后，发现这是一个新的 AI 使用方式，建立一个很好的规则，把人和LLM管理起来。

LLM 确实是非常好的知识工作者，人要训练很多年的技能，在LLM面前显得毫无价值。培养一个人写代码，起码需要10年时间，字母、逻辑、数学、计算机知识都是基础知识，有了这些基础知识才可能做计算机编码。所以培养一个程序员的时间对人类而言就是10多年，要想成为专家，在入门的之后，普通人依然需要花费10万小时。

而在人类的其它领域，比如培养一个合格的医生，从大学算起，依然需要差不多10年时间，培养科学工作者也需要差不多时间。人类进入工业文明以来，知识工作者一直是门槛极高的相对稀缺的工作。

但是随着 LLM 的出现，过去建立的知识工作者的优势瞬间消失。人们不再需要数十年时间来训练和培养一个知识工作者，掌握非常多的知识只是为了应付考试，甚至被戏称为小镇做题家。

在知识技能通用化和廉价的未来，这个未来很近，可能都不到2030年，90%的知识工作都变得廉价且随手可得。在这样的社会背景下，对个人而言，什么才是稀缺的技能或者资产。要把时间花在什么地方，学习什么东西才能在未来的生活中活得自由一点呢？

第一技能当然是充分理解知识且充分理解社会价值，有好的价值判断体系，有好的价值观。 懂得审美、懂得音乐、懂得和人正确的相处和交互。

然后需要深刻理解计算机，充分了解计算机的能力，能知道怎么样用计算资源为自己服务，为他人服务。满足自己的需求，为他人提供价值。在这样的基础上，充分发挥自己的兴趣。

人类将进入工业文明的尾声，开启机器智能文明。我现在无法判断人类是否会进入 Matrix，但是一定会因为人工智能的出现，整个社会发生重构。首先是生产力的疯狂提升，然后带来的问题是社会分配关系的重新调整。人类社会一定会面临物质过剩，然后物质充分极度丰富。人类的工厂将以需求数倍的产能，完全可以养活所有人。唯一的问题是如何有效分配。

这个过程不可逆，无法阻止。
